Funcionalidades clave

Este proyecto contiene las siguientes funcionalidades.

Registro y login

Usuarios con contraseña segura (hash) y sesiones.

Alta de actividades

Formulario con validación en cliente y servidor.

Historial

Listado por usuario con filtros básicos (tipo/fecha).

Fetch + JSON

Envío y carga dinámica de datos sin recargar la página.

Cómo funciona

Una arquitectura sencilla: HTML/CSS + JS (DOM) + PHP + MySQL.

  1. 1 Te registras e inicias sesión.
  2. 2 Añades actividades desde el dashboard.
  3. 3 PHP valida y guarda en MySQL con consultas preparadas.
  4. 4 El dashboard se actualiza con Fetch (JSON).
